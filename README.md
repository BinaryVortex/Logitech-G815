# ⌨️ Logitech G815

A premium web recreation inspired by the iconic Logitech G815 mechanical
gaming keyboard. This project focuses on delivering a visually stunning
product showcase with smooth interactions, elegant animations, and a
modern UI experience.

## ✨ Overview

The Logitech G815 project is a modern front-end experience inspired by
one of the most iconic low-profile mechanical gaming keyboards ever
created.
